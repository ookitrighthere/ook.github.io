## 2-5 소수(에라토스테네스 체)

### 간단한 문제 설명


1부터 N까지의 소수의 개수를 출력하기
```
예시 입력 👉 20
예시 출력 👉 8
```

<br>

### 내 답안


```java
public class Ex2_5 {
	public static void main(String[] args) {
		Scanner sc = new Scanner(System.in);
		int n = sc.nextInt();
		System.out.println(solution(n));
	}
	static int solution(int n) {
		int count = 0; 
		ArrayList<Integer> list = new ArrayList<>();
		for(int i = 1; i <= n; i++) {
			count = 0;
			for(int j = 1; j <= i; j++) { 
				if(i % j ==0) count++;			
			}	
			if(count==2) list.add(i);
		}
		return list.size();
	}
}

```

	static int solution(int n) {
