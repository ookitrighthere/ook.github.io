## [ Lv.1 ] 예산

예시 입력 👉 [1,2,3,4,5]

예시 출력 👉 [1]	


<br>

### 내 답안


```java
    public int solution(int[] d, int budget) {
		int answer = 0;
		Arrays.sort(d);
		for(int i = 0; i < d.length; i++) {
			budget -= d[i];
			if(budget >=0) answer++;
			else break;
		}
		return answer;
    }


```
- 신청금액이 작은 부서부터 지원해서 최대한 많은 부서를 지원하게 한다.

1. 부서별 신청금액 리스트(d)를 오름차순으로 정렬한다.
2. 예산에서 d를 0 이상이 될때까지 순차적으로 차감한다.
3. 차감하면 지원한 부서(answer)를 추가시킨다
